# Contributing to Endee

We welcome contributions from the community to help make vector search faster and more accessible for everyone.

## Ways to Contribute

- Submit pull requests for fixes, features, and improvements
- Report bugs or performance issues through GitHub Issues
- Propose enhancements for search quality, performance, and deployment workflows
- Improve documentation and examples

## Getting Started

1. Fork the repository on GitHub
2. Clone your fork locally
3. Create a feature branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Run the tests to ensure nothing is broken
6. Commit your changes: `git commit -m "feat: describe your change"`
7. Push to your fork: `git push origin feature/your-feature-name`
8. Open a Pull Request against the `master` branch

## Code Style

This project uses `clang-format` for C++ code formatting. Run:

```bash
clang-format -i <file>
```

Or use the `.clang-format` config at the repo root.

## Reporting Issues

Please use GitHub Issues to report bugs. Include:
- A clear description of the problem
- Steps to reproduce
- Expected and actual behaviour
- Your OS and CPU architecture
